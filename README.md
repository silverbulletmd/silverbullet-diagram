A pure [Space Lua](https://silverbullet.md) library that adds [Mermaid](https://mermaid.js.org/) diagrams to [SilverBullet](https://silverbullet.md).

It gives you two ways to draw diagrams:

* **Static diagrams** — ordinary fenced ` ```mermaid ` code blocks, live-previewed.
* **Dynamic diagrams** — a Space Lua API that generates Mermaid diagrams **at
 runtime from your space's data** (the object index). A diagram becomes just another live view of your notes: edit a page's frontmatter and the diagram redraws itself, exactly like a query table.

More details in the library itself: Mermaid.md

Requires: SilverBullet 2.10+ (or edge, as 2.10 is not yet released at the time of this writing).

# Installation
Open the SilverBullet "Library Manager" and install via the URI: `github:silverbulletmd/silverbullet-diagram/Mermaid.md`
